# axe

A complete open source e-commerce solution.

## Install jdk

```
curl -s "https://get.sdkman.io" | zsh
sdk install java
sdk install gradle
sdk default java 8u144-zulu
```

## Build

```
git clone https://github.com/kapmahc/axe.git
cd axe
gradle build
```
